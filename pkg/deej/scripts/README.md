### Windows (PowerShell)

- `windows\\build-dev.ps1`, `windows\\build-release.ps1`, `windows\\build-all.ps1` - build Windows binaries on Windows.
- `windows\\make-icon.ps1`, `windows\\make-rsrc.ps1` - utilities used to update resources.
- `windows\\prepare-release.ps1` - prepare github release.

### Windows (Linux cross-compilation)

- `windows\\build-dev.sh`, `windows\\build-release.sh`, `windows\\build-all.sh` - build Windows binaries from Linux

### Linux

- `linux\\build-dev.sh`, `linux\\build-release.sh`, `linux\\build-all.sh` - build Linux binaries on Linux
- `common.sh` - include file.
